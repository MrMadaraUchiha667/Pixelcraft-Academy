# 📚 Sobre a Pixelcraft Academy

A **Pixelcraft Academy** é a plataforma oficial de ensino da **Pixelcraft Studio**, criada para formar profissionais capazes de desenvolver, configurar e administrar servidores de jogos multiplayer.

Nosso objetivo é transformar jogadores apaixonados em profissionais qualificados para atuar em comunidades, empresas e projetos próprios.

Os cursos foram desenvolvidos para ensinar desde os conceitos básicos até técnicas utilizadas em grandes servidores.

# 🏆 Certificação

Ao concluir todos os requisitos do curso, o aluno poderá obter um certificado digital exclusivo da Pixelcraft Academy.

Cada certificado possui:

* Número de identificação
* Código de validação
* QR Code
* Data de emissão
* Carga horária
* Nome do curso
* Assinatura digital
